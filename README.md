# Homebrew Tap for Revenium

Official [Homebrew](https://brew.sh) tap for [Revenium](https://revenium.io) — the AI Economic Control System.

## Install
```bash
brew install revenium/tap/<formula>
```

Or tap first, then install:
```bash
brew tap revenium/tap
brew install <formula>
```

## Formulae

| Formula | Description |
|---|---|
| `revenium-cli` | Official CLI for the Revenium AI Economic Control platform |

## Why a tap?

Revenium distributes official signed binaries through this tap. Formulae are maintained by Revenium and updated automatically as new releases are published.
